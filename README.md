# AI-Based Autonomous Health Monitoring and Resource Management for PocketQube Satellites

This project develops a lightweight AI-based health monitoring system for PocketQube satellites using machine learning.

## Features

- Satellite telemetry analysis
- Random Forest anomaly detection
- PocketQube autonomous health report
- Resource management recommendations
- Power management
- Thermal management
- Communication optimization

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Git & GitHub
